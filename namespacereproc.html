Detailed Description

The reproc namespace wraps all reproc C++ declarations. reproc::process wraps the C api inside a C++ class. reproc::errc improves on REPROC_ERROR by integrating with C++'s std::error_code error handling mechanism. To avoid exposing the C API when using the C++ API all the other enums and constants of the C API have a replacement in the reproc namespace as well.
